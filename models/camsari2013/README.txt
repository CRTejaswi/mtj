************* Modular Library for Spintronics *****************
Version: 1.0.0alpha

This bundle contains 3 folders each containing the modules in MATLAB, HSPICE and Verilog-AMS languages.

For understanding the details and usage of this library see: https://nanohub.org/groups/spintronics

Please report any bugs, suggestions, comments at https://nanohub.org/groups/spintronics/forum

Note: This is an initial release. The three vesrions maynot be "API" compatible. They shall be made so in future releases.
